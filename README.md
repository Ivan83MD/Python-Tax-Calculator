# Python-Tax-Calculator

# Project Overview
This project is a Python-based tax calculator developed using multiple programming paradigms:
* Procedural Programming
* Object-Oriented Programming (OOP)
* Functional Programming
* Event-Driven Programming (GUI)

The application calculates tax based on income and displays a formatted payslip.
# Features
* Tax calculation based on income bands
* Multiple programming paradigms in one application
* User input validation
* Graphical User Interface (Tkinter)

# Technologies Used
* Python
* Tkinter (for GUI)
* PyCharm

# What I Learned
* Differences between programming paradigms
* Writing clean and structured code
* Debugging and testing applications
* Building simple GUI applications

# How to Run
1. Download the project
2. Open in PyCharm or any Python IDE
3. Run the file:
python tax_calculator.py
